# Shirts of Anime 
#### Basic image gallery showcasing T-shirts seen in anime
Requires:
 * node.js & npm
 * MySQL/MariaDB
 * ImageMagick

#### How to deploy:
1. Create `shirtsofanime` database from MySQL console
2. `npm install`
3. `bower install`
4. Edit `config.json` with desired config
5. `node server.js`
